# MOBBING

Kurumsal hayatta kalma simülasyonu — Reigns usulü kart oyunu.
Barış'ın gerçek mobbing deneyiminden doğan hiciv/farkındalık oyunu.

- **iOS:** %100 native SwiftUI, iOS 26 Liquid Glass (`ios/`)
- **Android:** %100 Kotlin + Jetpack Compose (`android/`)
- **Kartlar:** Leymann LIPT-45 tabanlı, çift dilli JSON (`cards/`)
- Ücretli, reklamsız, çevrimdışı, veri toplamaz.
- Privacy: https://realvirtuality.app/mobbing/privacy.html
